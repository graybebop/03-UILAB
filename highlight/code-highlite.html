html highlight를 쓰기위한 조건

1. 아래 링크 헤더에 삽입(default.min.css -> agate.min.css 변경해서 사용)

    <!--<link rel="stylesheet" href="//cdnjs.cloudflare.com/ajax/libs/highlight.js/9.5.0/styles/default.min.css">-->
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/highlight.js/11.1.0/styles/agate.min.css">
    <script src="//cdn.jsdelivr.net/highlight.js/9.5.0/highlight.min.js"></script>
    <script>
        hljs.initHighlightingOnLoad();
    </script>
2. HTML태그를 특수태그로 변환
3. https://highlightjs.org/static/demo/ 에서 원하는 스타일로 styles/agate.min.css에서 agate이름을 변경 적용하면 된다.
