EM에 관해서

길이가 유연한 가변 단위. media query와 함께 사용하면, 뷰포트의 크기에 따라 길이를 가변적으로 핸들링 할 수 있습니다. 상위 엘리먼트의 폰트 사이즈를 기준으로 적용되는 방식

parent클래스가 25px, child클래스는 1.5em child는 parent의 px의 영향을 받아서 폰트사이즈가 커지고 작아진다.
폰트사이즈 (parent)
폰트사이즈 (child)
                        
.parent {
    font-size: 25px;
}

.child {
    font-size: 1.5em;
}
parent2클래스가 15px, child클래스는 1.5em child2는 parent2의 px의 영향을 받아서 폰트사이즈가 커지고 작아진다.
확실히 차이나는것을 볼 수 있다.

폰트사이즈 (child)

}

}
.parent6 {
